## 🚀 Features

- ➕ Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division
- ✅ Clear (`C`) and Equals (`=`) buttons
- 👀 Live input and result display
- 📐 Built with C++/CLI using Windows Forms
